Preguntas Frecuentes (FAQ)

Bienvenido a la sección de Ayuda de Flow Manager. Aquí encontrarás respuestas a las preguntas más frecuentes sobre el uso de nuestra plataforma, sus funcionalidades y cómo puedes sacarle el mayor provecho. Si tienes alguna duda adicional o necesitas asistencia, no dudes en contactarnos a través de nuestros canales de soporte. ¡Estamos aquí para ayudarte!

1. ¿Qué es Flow Manager?

Flow Manager es una plataforma digital diseñada para facilitar la gestión de empleados y optimizar procesos administrativos, como la solicitud de licencias, la administración de días libres y el envío de documentos importantes.

2. ¿Cómo puedo acceder al sistema?

Puedes acceder a Flow Manager desde cualquier dispositivo con conexión a internet. Solo necesitas ingresar con tu usuario y contraseña proporcionados por tu empresa.

3. ¿Es seguro el sistema?

Sí, Flow Manager implementa medidas de seguridad avanzadas, como cifrado de datos y autenticación de usuarios, para proteger la información sensible de empleados y empresas.

4. ¿Qué puedo gestionar como empleado?

Los empleados pueden solicitar licencias, ver sus días libres disponibles, recibir documentación oficial y comunicarse directamente con el departamento de Recursos Humanos a través de la plataforma.

5. ¿Qué funcionalidades ofrece Flow Manager a las empresas?

Flow Manager permite a las empresas gestionar permisos, enviar documentos, organizar reuniones, y supervisar las operaciones internas de forma eficiente y centralizada.

6. ¿Cómo se puede contratar el servicio?

Puedes contratar Flow Manager contactando a nuestro equipo a través del formulario de contacto en nuestro sitio web en la sección "Hazte cliente". Ofrecemos diferentes planes según las necesidades de cada empresa.

7. ¿Cuáles son los requisitos técnicos para utilizar el sistema?

Solo necesitas un dispositivo con acceso a internet y un navegador web actualizado. Flow Manager es compatible con la mayoría de los navegadores modernos.

8. ¿Es necesario descargar algún software?

No, Flow Manager es completamente accesible desde la web, por lo que no necesitas descargar ni instalar ningún programa adicional.

9. ¿Cómo se gestionan los roles y permisos dentro del sistema?

El sistema permite asignar roles específicos a cada usuario, como empleados, directores o personal de Recursos Humanos, asegurando que cada uno tenga acceso solo a las funciones necesarias para su rol.

10. ¿Qué sucede si tengo problemas para acceder al sistema?

Si tienes problemas para acceder, puedes comunicarte con nuestro equipo de soporte técnico a través del correo electrónico o el número de contacto disponible en la plataforma.
